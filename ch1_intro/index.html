⚛️ React Chapter 1: Introduction

📌 What is React?

React is a JavaScript library used to build user interfaces. It updates the UI automatically when data changes. 

Normal JS:
document.getElementById("root").innerText = "Hello";

React:
<h1>Hello</h1>

📌 JSX

JSX lets you write HTML inside JavaScript.

const name = "Najma";

const element = <h1>Hello {name}</h1>;

📌 Components

Components are reusable UI functions.

function App() {
  return <h1>My First Component</h1>;
}

root.render(<App />);
